# 云端部署指南 - GitHub Actions

## 已完成的步骤

✓ Git 仓库已初始化
✓ GitHub Actions 工作流已配置
✓ .gitignore 已设置

## 需要你完成的步骤

### 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称：`daily-news-bot`
3. 设为**公开**或**私有**均可
4. **不要**初始化 README、.gitignore 等
5. 点击"创建仓库"

### 步骤 2：推送代码到 GitHub

```bash
cd "/Users/mac/Desktop/claudecode项目/每日新闻推送系统"
git remote add origin https://github.com/你的用户名/daily-news-bot.git
git branch -M main
git push -u origin main
```

### 步骤 3：配置 GitHub Secrets

1. 访问仓库的 Settings → Secrets and variables → Actions
2. 点击"New repository secret"
3. 添加以下 4 个密钥：

| 名称 | 值 |
|------|------|
| `FEISHU_WEBHOOK` | `https://open.feishu.cn/open-apis/bot/v2/hook/b1d55080-ee95-44fe-b73d-f2484736bcff` |
| `ANTHROPIC_API_KEY` | `4806a297af204a0fb2316e8f5e0bae3d.lPXVH3sUG1MYPySI` |
| `ANTHROPIC_BASE_URL` | `https://open.bigmodel.cn/api/anthropic` |
| `ANTHROPIC_MODEL` | `glm-4.7` |

### 步骤 4：启用 GitHub Actions

1. 访问仓库的 Actions 页面
2. 确认工作流已启用
3. 点击"Run workflow"手动测试

### 步骤 5：验证运行

1. 查看运行日志：Actions → 最新运行
2. 确认飞书收到消息

---

## 定时任务说明

- **触发时间**：每天 UTC 1:30（北京时间 9:30）
- **手动触发**：Actions → Daily news bot → Run workflow
- **运行日志**：保留 7 天

## 一键部署命令

完成步骤 1 后，运行以下命令：

