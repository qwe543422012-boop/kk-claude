# 每日新闻推送系统 - 使用说明

## 快速命令

### 手动运行
```bash
cd "/Users/mac/Desktop/claudecode项目/每日新闻推送系统"
python3 run.py
```

### 快速测试（每类 1 条）
```bash
python3 test.py
```

### 查看日志
```bash
tail -f logs/cron.log
```

### 查看定时任务
```bash
crontab -l
```

### 修改定时任务时间
```bash
crontab -e
# 格式：分 时 日 月 周
# 例如：每天 9:30 → 0 9 * * *
#      每天 10:00 → 0 10 * * *
```

### 停用定时任务
```bash
crontab -r  # 删除所有定时任务
```

## 系统信息

- **运行时间**：每天 9:30
- **新闻数量**：每类 10 条
- **AI 服务**：智谱 GLM-4.7
- **推送渠道**：飞书群聊
